**Trendora** is a full-stack e-commerce web application built with the **MERN stack**, designed for a modern clothing brand. Users can browse products, filter by category/size/price, manage their cart, and complete a mock checkout with order confirmation emails.

---

## âœ¨ Features

- **User Accounts & Authentication**  
  - Register & login with **secure password hashing**  
  - JWT-based authentication for protected routes  

- **Product Catalog**  
  - Browse and filter clothing items by category, size, and price  
  - Product details page with images, descriptions, and sizes  

- **Shopping Cart**  
  - Add/update/remove items with size selection  
  - Guest cart support (add items without login)  

- **Checkout & Orders**  
  - Mock checkout process  
  - Orders saved in **MongoDB**  
  - Confirmation email via **Nodemailer**  

- **Frontend**  
  - React (Vite) with **react-router-dom** and **react-hot-toast**
